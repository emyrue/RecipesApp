# Recipe App

> The Recipe App allows users to create accounts where they can create their own recipes, manage their inventory and create shopping lists. The application also allows them to add foods to the site so that they can be added to recipes or inventories.


## Built With

- Ruby on rails
- Postgresql

## Getting Started

To get a local copy up and running follow these simple example steps.

- Navigate to your desired directory
- Type the following commands into your terminal:
    - git clone git@github.com:emyrue/RecipesApp.git
    - cd RecipesApp
    - rails db:create
    - rails db:migrate

### Running Tests

Run the following commands in your terminal:
```
bundle install
```
```
rspec
```
